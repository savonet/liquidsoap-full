Module Vorbis.Skeleton

module Skeleton: sig .. end
val fisbone : ?start_granule:Stdlib.Int64.t ->
       ?headers:(string * string) list ->
       serialno:Stdlib.Nativeint.t ->
       samplerate:Stdlib.Int64.t -> unit -> Ogg.Stream.packet

Generate a vorbis fisbone packet with these parameters, to use in an ogg skeleton. Default value for start_granule is Int64.zero, Default value for headers is "Content-type","audio/vorbis"

See: http://xiph.org/ogg/doc/skeleton.html.
