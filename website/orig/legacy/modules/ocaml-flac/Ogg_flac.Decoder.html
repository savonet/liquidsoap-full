Module Ogg_flac.Decoder

module Decoder: sig .. end

Decode ogg/flac data

Usage

Usage is similar to the case of the native FLAC decoder, using the appropriate ogg/flac decoding callbacks.

The main difference is that in the case of the ogg/flac decoding, the exception Ogg.Not_enough_data may be raised if the ogg stream used to create the decoder does not contain enough data. In this case, you should feed more data into the ogg stream and call the decoding function again.

This remark is valid for both the Flac.Decoder.init and Flac.Decoder.process functions.
