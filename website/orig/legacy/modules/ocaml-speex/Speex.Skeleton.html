Module Speex.Skeleton

module Skeleton: sig .. end
val fisbone : ?start_granule:Stdlib.Int64.t ->
       ?headers:(string * string) list ->
       serialno:nativeint -> header:Speex.Header.t -> unit -> Ogg.Stream.packet

Generate a vorbis fisbone packet with these parameters, to use in an ogg skeleton. Default value for start_granule is Int64.zero, Default value for headers is "Content-type","audio/speex"

See: http://xiph.org/ogg/doc/skeleton.html.
