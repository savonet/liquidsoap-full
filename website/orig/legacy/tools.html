Savonet : Tools using Liquidsoap

Here are some tools which are using Liquidsoap in order to provide interfaces, schedulers or web integration. Feel free to contact us if you are aware of some more!

  • Airtime lets you take total control of your radio station via the web with intelligent archive management, powerful search, an easy playlist builder, a simple scheduling calendar and rock-solid automated playout. Those who need a little extra will love the ability to manage staff, record and rebroadcast a live show, upload to SoundCloud automatically, stream to Icecast and display programme information via Airtime's website widgets. It is written in Python/Django.
  • AzuraCast is a lightweight “all-in-one” turnkey radio station management kit, with modular support for radio backends and frontends.
  • Bufferplot displays graphically the buffer levels from logs.
  • Def Playlist Manager is a browser-based playlist manager running on MySQL/PHP.
  • FileScheduler selects in your contents the best next one. No complex interface, no configuration file, simply files and directories.
  • Liquidsoap Requester provides an extremely-lightweight and reliable web control interface for the Liquidsoap Multimedia Stream Generator software.
  • Liquidsoap:Scheduler can be used to dynamically switch between multiple audio input streams. Its designed to run fulltime without any user interaction except synchronization with an external Event Calendar like Google Calendar. It is written in PHP.
  • Robinet is a GUI based on Liquidsoap to send a stream to multiple servers. It is written in Python/Gtk.
  • Shampoo is an all-in-one web platform that allows you to create, manage, and deliver to streaming servers and listeners your own webradios and radio shows. It is written in Java.
  • Washtub is a web-based liquidsoap stream controller that attempts to interface with as many telnet commands as possible. It is written in Python/Django.
