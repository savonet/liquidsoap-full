Liquidsoap on OS X using Homebrew

You can download, build and install liquidsoap from source on Mac OS X using Homebrew. Here are the instructions.

  • Install Homebrew
  • Enable drfill/liquidsoap 
brew tap drfill/liquidsoap
  • Install Liquidsoap:
brew install liquidsoap
  • Test it :)
osxbox:~ osxbox$ liquidsoap --version
